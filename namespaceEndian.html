Endianess Endian::diff ( Endianess  e1,
Endianess  e2 
)
inline

Returns what endianess operation is needed to convert between two endianesses. The returned endianess represents the operation needed to be done between host system's endianess and returned one, to achieve the same conversion as between the two given endianesses. For example, will return Endianess::NATIVE if no conversion is needed.
